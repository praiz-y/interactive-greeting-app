# Interactive Greeting App

A simple interactive web application built with **HTML, CSS, and Vanilla JavaScript** that generates personalized greetings based on user input and time of day.

This project was created to practice core JavaScript concepts such as DOM manipulation, event handling, input validation, and dynamic UI updates.

---

## Features

- Personalized greeting using user input
- Time-based greetings (Morning, Afternoon, Evening)
- Random welcome messages
- Input validation with visual feedback
    - Reset functionality
- Dynamic error messages
- Interactive button hover effects

---

## Concepts Practiced

- DOM Selection & Manipulation
- Event Listeners
- Functions & Arrow Functions
- Conditional Logic
- Arrays & Random Selection
- Form Validation
- CSS Class Toggling
- User Interaction Handling

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
